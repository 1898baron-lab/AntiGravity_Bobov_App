## 1. Контекст проекта
- **Изделие**: Сборочно-сварочная оснастка (ССО / Стапель).
- **Сборочная единица**: Рама 1200x800 мм, профильная труба 40x40x3 (Сталь 20).
- **Этап**: Разработка концепции базирования и закрепления.
- **Ограничения**: Необходим доступ для MAG-135 сварки всех внутренних и внешних углов.

## 2. Текущая конструкция / Решение
- **Схема базирования**: 3-2-1.
- **Опоры**: Винтовые регулируемые (3 шт.) под плоскость.
- **Упоры**: Съемные пальцы (3 шт.) по контуру.
- **Зажимы**: Toggle Clamps (быстросъемные рычажные).

## 3. Проблема / Неопределенность
Требуется подтверждение точности сборки и оценка рисков термических деформаций при сварке MAG-135. Есть сомнение в эквивалентности материала S235 российским нормам ЦПТИ для АЭС.

## 4. Конкретные вопросы
1. Соответствует ли точность рамы ±1.0 мм (диагонали ±1.5 мм) **12-му квалитету по ГОСТ 30893.1**?
2. Какие риски по деформациям ты видишь при сварке трубы стенкой 3 мм в таком габарите? Нужен ли антидеформационный порядок сварки?
3. Является ли сталь **S235** полным эквивалентом **Стали 20** для конструкций, работающих под регламентом Росатома?

## 5. Ожидаемый результат
- Подтверждение/корректировка квалитета.
- Рекомендации по последовательности наложения швов.
