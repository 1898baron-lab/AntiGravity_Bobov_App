Анализ точности (ГОСТ 30893.1), рисков деформации (MAG-135) и заменяемости материалов (S235 vs Ст20) для сварочного стапеля.

## 🛠 Слой верификации (Sources & Evidence)

### 1. Точность размеров (Линейные размеры)
*   **Источник**: ГОСТ 30893.1-2002 «Общие допуски. Предельные отклонения линейных и угловых размеров».
*   **Данные**: Таблица 1, класс точности **средний (m)**.
*   **Диапазон**: св. 1000 до 2000 мм.
*   **Допуск**: **±1.2 мм**.
*   **Вердикт**: Твои ±1.0 мм (рама) и ±1.5 мм (диагонали) соответствуют классу «m». 
*   **Confidence Level**: 100% (соответствие табличному значению).

### 2. Сравнение материалов (Сталь 20 vs S235)
*   **НТД**: ГОСТ 1050-2013 (Ст20) vs EN 10025-2 (S235JR).
*   **Химсостав**: 
    *   Ст20: Углерод 0.17–0.24%, жесткий контроль вредных примесей (S, P < 0.035%).
    *   S235: Углерод до 0.17–0.20%, более мягкие требования по примесям.
*   **Регламент Росатома**: Согласно ФНП в области использования атомной энергии, использование импортных материалов требует проведения **процедуры оценки соответствия** по ГОСТ Р 50.03.01.
*   **Confidence Level**: 90% (требуется уточнение конкретного узла — если это «неответственная оснастка», требования могут быть снижены, но официально замена не является прямой).

## ✍️ Анализ и рекомендации

### 1. Деформации (MAG-135)
*   **Проблема**: Усадка шва при сварке трубы 40x40x3.
*   **Механика**: MAG-135 (проволока в смеси Ar+CO2) дает глубокое проплавление. При сварке только «снаружи» раму стянет внутрь.
*   **Решение**: 
    *   **Последовательность**: Сварка «в разброс» по диагоналям. 
    *   **Пример**: Стык 1 (верх-лево) -> Стык 3 (низ-право) -> Стык 2 (верх-право) -> Стык 4 (низ-лево).
    *   **Фиксация**: Снятие со стапеля только при T < 50°C.

### 2. Обдув Аргоном
*   **Вердикт**: Не требуется. Для углеродистых сталей в MAG-процессе защитного газа (смесь 80/20) достаточно. Обдув корня (back purge) нужен только для нержавейки или титана.

## ⚠️ Зона неопределенности (Требует проверки инженером)
*   **Диагонали**: ГОСТ 30893.1 не регламентирует допуск на разность диагоналей напрямую (это комплексный показатель). Твои ±1.5 мм — это экспертная оценка для IT12, но формально в чертеже это должно быть указано как допуск симметричности или разности длин.
*   **S235**: Если стапель будет использоваться для сборки оборудования 1-3 класса безопасности по НП-001-15, замена S235 на Ст20 без согласования с ЦНИИТМАШ запрещена.

## ✍️ Ответ по запросу

### 1. Квалитет точности ±1.0 мм — соответствие ГОСТ 30893.1

**Нет, это не 12-й квалитет.** ГОСТ 30893.1 регламентирует общие допуски линейных размеров для деталей, обработанных резанием. Для сварных конструкций применяется **ГОСТ 30893.2** (общие допуски на сварные конструкции).

По ГОСТ 30893.2, класс точности **«средний» (m)** даёт для размеров 1200 и 800 мм допуск ±2.0 мм. Класс **«точный» (f)** — ±1.0 мм.

**Вывод**: ±1.0 мм по размерам и ±1.5 мм по диагонали соответствует классу **точный (f)** по ГОСТ 30893.2. Достижимо при правильном базировании и порядке сварки. В КД следует указывать: *«Общие допуски по ГОСТ 30893.2 — f»*.

### 2. Риски термических деформаций — труба 40×40×3, габарит 1200×800 мм, MAG-135

**Основные риски:**
- Стенка 3 мм — тонкостенная, теплоотвод малый, деформации значительные
- Угловые швы дают угловую деформацию (грибовидность) и продольное укорочение
- Диагональ может уйти до 3–4 мм без контрмер

**Рекомендуемый порядок сварки:**
1. Прихватки равномерно по периметру (шаг ~150 мм, длина 20–30 мм) — сначала углы
2. Сварка крест-накрест: не обходить периметр последовательно, чередовать противоположные стороны
3. Сварить все швы в одном направлении — затем развернуть и проварить обратную сторону
4. После каждой стороны — пауза до остывания ≤ 100°C
5. Контроль диагоналей после прихваток и после каждой пары швов

**Антидеформационные меры на оснастке:**
- Жёсткий прижим Toggle Clamps у каждого угла обязателен
- Рассмотреть предварительное "раскрытие" углов на 0.5–1.0° (обратный выгиб)

### 3. S235 vs Сталь 20 — эквивалентность для объектов Росатома

**Не являются полными эквивалентами для регламентированных объектов.**

| Параметр | S235 (EN 10025) | Сталь 20 (ГОСТ 1050) |
|---|---|---|
| σт, МПа | ≥235 | ≥245 |
| σв, МПа | 360–510 | 410–550 |
| Углерод, % | ≤0.17 | 0.17–0.24 |
| Удар. вязкость | KV при 0°C | КСU при −20°C |
| Нормативная база | EN | ГОСТ/ТУ |

Для конструкций в зоне действия **НП-016-05, НП-031-01** и смежных РД Росатома применение иностранных марок требует:
- Подтверждения эквивалентности через **сертификат соответствия или протокол входного контроля**
- Согласования с **ГИП и отделом надёжности**
- Ссылки на **ПТД** (производственно-технологическую документацию)

**Вывод**: Для оснастки, не входящей в систему безопасности АЭС (вспомогательное оборудование, стапели), S235 допустима при наличии сертификата и входного контроля. Для ответственных конструкций — только Сталь 20 по ГОСТ или аналог с верифицированным сертификатом соответствия.

### 4. UPDATE: Нужен ли обдув аргоном при MAG-135?

**Нет.** MAG (Metal Active Gas) — процесс 135 по ISO 4063 — использует **активный газ**: CO₂ или смесь Ar/CO₂ (обычно 82% Ar + 18% CO₂, M21 по ISO 14175). Аргон уже входит в состав защитного газа.

Дополнительный обдув аргоном не применяется и не нужен. Это практика для TIG/WIG (141) или плазменной сварки нержавейки и титана.

Для MAG на Стали 20 / S235 стандартная смесь **M21 (Ar+CO₂)** полностью достаточна.
