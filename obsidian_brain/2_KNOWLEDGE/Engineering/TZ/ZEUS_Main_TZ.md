# 📐 Техническое Задание: Проект "ZEUS" (КУПЕ)
**Наименование:** Разработка комплексной установки для перезарядки емкостей (КУПЕ)
**Шифр:** 10.129.0.00.00.000.000 ТЗ

---

## 1. Основание для разработки
- **Договор:** № 39/12/21002-Д/311/6310-Д от 19.12.2024.
- **Заказчик:** АО «УЭХК».
- **Исполнитель:** АО «ЦПТИ».

## 2. Цель и назначение
### 2.1 Цель
Создание мобильного устройства на современной элементной базе для дистанционного вскрытия (сверления) заглушек емкостей с ОГФУ объемом 1.0 м³ и 2.5 м³.

### 2.2 Назначение
- Гарантированный сброс избыточного давления.
- Удаление взрывоопасных смесей газов (UF6, HF, F2, H2).
- Работа с "глухими" заглушками и заглушками с клапаном Ду3.

## 3. Технические характеристики
| Параметр | Значение |
| :--- | :--- |
| **Способ сверления** | Пневматический |
| **Время установки** | не более 600 сек |
| **Диаметр отверстия** | не менее 8 мм |
| **Макс. давление в камере** | до 0.17 МПа |
| **Масса устройства** | **не более 25 кг** (без шкафа управления) |
| **Напряжение (датчики)** | 12 В |
| **Срок службы** | 10 лет |

## 4. Состав изделия (Основные узлы)
1. **Площадка мобильная (1)** — база для размещения элементов.
2. **Защитная камера (2)** — герметичная зона для изоляции газов.
3. **Пневмопривод (6)** — обеспечивает вертикальное перемещение дрели.
4. **Пневмодрель (4)** — режущий инструмент (ход не менее 40 мм).
5. **Шкаф управления (5)** — контроллеры и КИПиА (удаление до 15-25 м).
6. **Система КИПиА** — датчики давления и газоанализатор водорода.

## 5. Технологический процесс (Сверление)
1. **Подготовка:** Установка прокладки, центрирование на горловине.
2. **Герметизация:** Вакуумирование камеры, напуск азота.
3. **Сверление:** Дистанционный запуск дрели. Контроль давления и водорода.
4. **Промывка:** Воздушная (азотная) промывка объема емкости.
5. **Финиш:** Автоматическое отключение и возврат дрели в верхнее положение.
