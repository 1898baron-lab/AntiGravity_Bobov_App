# Схема сварочного стапеля: Рама 1200х800 (Вид сверху)

Схема фиксации для предотвращения деформаций при сварке MAG 135.

## 🔲 ASCII-Модель стапеля

```text
    <--- 1200 мм --->
   _______________________
  | [F1]             [F2] |  [F1...F4] - Винтовые прижимы M16
  |  L                 J  |  [L, J, Г, 7] - Угловые упоры (Базы)
  |                       |
  |                       |
  |                       |  == [W] == Зоны сварки (углы)
  |  Г                 7  |
  | [F4]             [F3] |
  |_______________________|
```

## 📍 Спецификация элементов оснастки:
1. **Угловые упоры (4 шт.)**: Выполнены из калиброванного уголка (Сталь 45). Обеспечивают перпендикулярность углов рамы.
2. **Винтовые прижимы (4 шт.)**: Прижимное усилие **5000 Н** на точку. Обеспечивают плотный контакт заготовки с упорами.
3. **Опорные платики (8 шт.)**: Расположены под профилем рамы. Имеют регулировку по высоте (набор прокладок) для компенсации неровности пола в цеху.

---
## 🏁 Технологические указания:
- **Базирование**: Деталь устанавливается "на упор" внешним контуром. 
- **Доступ**: Конструкция стапеля обеспечивает свободный доступ горелки к внешним и внутренним углам рамы.
- **Остывание**: ЗАПРЕЩАЕТСЯ снимать раму со стапеля до полного остывания швов (до T < 50°C), во избежание "увода" геометрии.

*Схема спроектирована: Mastodont Engineering Module (ПК-5).*
