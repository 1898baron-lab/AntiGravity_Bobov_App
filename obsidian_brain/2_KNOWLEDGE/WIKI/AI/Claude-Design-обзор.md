Claude Design — обзор возможностей
Три реальных кейса: от прототипа мобильного приложения до питч-дека для инвестора и лендинга SaaS-
продукта. Всё создается через текстовые промпты с помощью нового инструмента Anthropic — Claude 
Design.
ANTHROPIC CLAUDE DESIGN 3 ДЕМО
Демо #1 — Прототип мобильного приложения
Роль: Ты — Senior UX/UI Designer с опытом в HR-Tech SaaS.
Задача: Создай высокодетализированный прототип мобильного приложения для платформы 
Connectio — сервиса AI-рекрутинга.
Контекст:
• Connectio — HR-платформа, которая автоматизирует оценку кандидатов через AI
• Кандидат проходит 9 валидированных тестов, после чего AI выдаёт вердикт: нанять / не 
нанять
• Целевая аудитория: HR-специалисты и руководители малого/среднего бизнеса
• Бренд: профессиональный, тёмная тема, акцентный цвет — синий (#2563EB)
Что создать:
1. Экран дашборда с активными вакансиями и количеством кандидатов на каждую
2. Экран профиля кандидата с радарной диаграммой результатов тестов
3. Экран AI-вердикта: рекомендация нанять/не нанять с обоснованием
Стиль: Clean tech-premium, минималистичный, мобильный формат (390x844). Интерактивность: 
кнопки переключения между экранами, возможность нажать на кандидата и увидеть профиль.
Ограничения: Без generic HR-иконок, без stock photo. Все данные — реалистичные (русские 
имена, реальные должности, конкретные процентные показатели тестов). 
Демо #2 — Презентация для Yoolip AI
Роль: Ты — Pitch Deck Strategist для SaaS-стартапов на ранней стадии. Задача: Создай investor 
pitch deck на 10 слайдов для Yoolip AI.
Контекст:
• Yoolip AI — SaaS-платформа для аналитики рекламы на маркетплейсе Ozon
• Помогает селлерам оптимизировать рекламный бюджет через AI-анализ кампаний
• Стадия: pre-seed, ищем первый раунд инвестиций
• Бренд: изумрудно-зелёный (#0E9E5A) как основной, тёмный фон (#0d1117), шрифт — Inter
Структура слайдов:
1. Титульный: название + one-liner + логотип
2. Проблема: селлеры тратят 30-50% бюджета впустую, не понимая какая реклама работает
3. Решение: AI-аналитика, которая в реальном времени показывает ROI каждой кампании
4. Рынок: TAM российского e-commerce advertising — $X млрд
5. Продукт: скриншот дашборда (сгенерируй реалистичный UI)
6. Бизнес-модель: подписка $29-99/мес по тарифам
7. Traction: первые клиенты, метрики
8. Конкуренты: матрица сравнения
9. Команда
10.Ask: раунд, использование средств
Стиль: Premium dark SaaS aesthetic. Минимум текста, максимум визуальных данных. Каждый слайд 
— одна ключевая мысль. Ограничения: Без stock-фото. Без мотивационных цитат. Без слова 
«революционный». Данные должны выглядеть реалистично.
Демо #3 — Лендинг 
Роль: Ты — Landing Page Designer для SaaS-продуктов. Задача: Создай landing page для Claude 
Design — воображаемый лендинг, который объясняет продукт Anthropic простыми словами для 
русскоязычной аудитории.
Структура:
1. Hero: заголовок «Дизайн без дизайнера» + подзаголовок + CTA-кнопка
2. Три преимущества: Прототипы, Презентации, Handoff в код
3. Секция «Как это работает»: 3 шага с иконками
4. Social proof: цитаты Brilliant и Datadog
5. CTA footer
Стиль: Тёмная тема, акцент — amber/orange (#D97706), шрифт — Inter. Минимализм. Никакого 
generic AI-арта (нейроны, роботы, матрица). Формат: Responsive, mobile-first. Интерактивный — 
кнопки, hover-эффекты, smooth scroll.
