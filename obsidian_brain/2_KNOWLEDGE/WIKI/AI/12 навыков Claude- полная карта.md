> [!abstract] ИИ Выжимка
> Статья описывает трансформацию Claude из простого чат-бота в мощную рабочую систему, способную автоматизировать сложные процессы. Ключевыми элементами этой эволюции являются кастомные «Skills» — постоянные инструкции для модели, и Model Context Protocol (MCP). MCP расширяет возможности Claude, предоставляя ему доступ к внешним системам, таким как почта, календарь, Notion и браузеры. Благодаря этим интеграциям, Claude становится полноценным цифровым ассистентом, способным не только отвечать на вопросы, но и инициировать действия в реальном мире.

Claude в 2026: от чат-бота 
к рабочей системе
12 навыков, которые меняют всё — без единой строчки кода. Эта трансформация делает Claude не 
просто диалоговой системой, а мощным, интегрированным инструментом, способным 
автоматизировать сложные процессы и выполнять задачи, требующие взаимодействия с 
внешними системами.
Skills
Skills — это текстовые инструкции для 
Claude, которые он запоминает навсегда, 
подобно должностной инструкции для 
сотрудника. Достаточно настроить их один 
раз, и Claude будет выполнять задачи в 
соответствии с заданными правилами и 
сценариями, значительно повышая свою 
автономность и эффективность.
MCP (Model Context Protocol)
MCP расширяет возможности Claude, 
предоставляя ему "руки" для 
взаимодействия с внешним миром. Это 
подключение к почте, календарю, Notion, 
браузеру и множеству других 
инструментов. Благодаря MCP, Claude 
выходит за рамки обычного чата, становясь 
полноценным цифровым ассистентом, 
способным инициировать действия и 
обрабатывать информацию из различных 
источников.
12 навыков Claude: полная карта
Skills и MCP-коннекторы, которые превращают Claude в рабочую систему.
01
Skills (кастомные инструкции)
Настраиваются в разделе Customize → Skills. Это текстовые инструкции, которые Claude запоминает навсегда, работая по заданным правилам и сценариям.
02
Memory MCP
Обеспечивает долгосрочную память между диалогами, позволяя Claude сохранять контекст и информацию для последующих взаимодействий.
03
Deep Research
Встроенный инструмент глубокого исследования — как ChatGPT Deep Research, но внутри Claude. Идеально для создания новых Skills и аналитики по любой теме. Не требует установки.
04
NotebookLM + Claude
Коннектор, предназначенный для эффективной работы с обширной базой знаний, интегрируя возможности NotebookLM с интеллектом Claude.
05
Google Workspace
Интеграция с Gmail, Google календарём и Google диском позволяет Claude управлять почтой, расписанием и файлами, становясь полноценным ассистентом.
06
Context7 MCP
Обеспечивает доступ к актуальной документации по библиотекам, таким как Next.js, Python и другим, для точной и релевантной информации.
07
FireCrawl MCP
Парсинг веб-сайтов конкурентов и преобразование данных в формат Markdown для быстрого анализа и получения структурированной информации.
08
Notion MCP
Агент Claude способен вести задачи и управлять базой знаний непосредственно в Notion, организуя рабочие процессы.
09
Playwright MCP
Позволяет автоматизировать действия прямо в браузере, расширяя возможности Claude для выполнения комплексных задач в интернете.
10
Gamma MCP
Создание презентаций на основе диалога с использованием предварительно заданных шаблонов, что упрощает и ускоряет процесс подготовки материалов.
11
FrontEnd Design Skill
Навык для создания визуально выдающихся веб-интерфейсов без "AI-слопа", обеспечивая качественный и чистый дизайн.
12
Skills Creator
Встроенный навык, позволяющий создавать новые Skills через Deep Research, расширяя функциональность Claude с минимальными усилиями.
НАВЫК 1 ИЗ 12
Skills — кастомные инструкции
Должностная инструкция для Claude. Один раз настроил — работает всегда.
Открой Claude Desktop
Нажми на иконку профиля → Customize
Перейди в раздел Skills
Нажми «Create new skill»
Заполни: название, описание (ключевые слова для вызова), инструкцию
Сохрани — Skills активен во всех чатах
Совет: в описании пиши фразы, которые ты реально используешь. Например: «мне нужно 
превью для YouTube» — и Claude сам найдёт нужный Skills.
Установка: встроено в Claude. Дополнительных инструментов не требуется.
НАВЫК 2 ИЗ 12
Memory MCP — долгосрочная память
Claude запоминает всё между сессиями. Больше не нужно повторять контекст.
Открой Claude Desktop
Перейди в Developer → Edit Config (claude_desktop_config.json)
Добавь блок Memory MCP-сервера в конфиг
Сохрани файл и перезапусти Claude Desktop
В новом чате проверь: напиши «Видишь ли ты Memory MCP?»
Готово — Claude теперь сохраняет факты между диалогами
Код блока Memory MCP-сервера:
{  "mcpServers": {    "memory": {      "command": "npx",      "args": ["-y", "@modelcontextprotocol/server-memory"]    }  }}    
Совет: скажи Claude «Запомни, что я предпочитаю краткие ответы без воды» — и он будет учитывать это в каждом следующем диалоге.
Требуется: Claude Desktop + Node.js
НАВЫК 3 ИЗ 12
Deep Research — глубокое исследование внутри Claude
Claude исследует тему как ChatGPT Deep Research. Бесплатно, внутри приложения.
Открой Claude Desktop или браузер (claude.ai)
Начни новый диалог — нажми на иконку «+» (New Chat)
В поле ввода найди иконку с лупой или выбери режим «Research» (рядом с кнопкой 
отправки)
Введи запрос — например: «Собери всё необходимое для создания скилла по 
высококонверсионному лендингу для SaaS»
Нажми отправить — Claude сам проведёт многоэтапное исследование и выдаст 
структурированный результат
Совет: Deep Research в Claude — это то же самое, что ChatGPT Deep Research и Google 
Deep Research. Называется просто «Research», поэтому легко пропустить. Идеально для 
создания новых Skills через Skills Creator.
Установка: встроено в Claude. Работает в браузере и Desktop. Дополнительных коннекторов не 
требует.
НАВЫК 4 ИЗ 12
NotebookLM + Claude — база знаний под рукой
Claude обращается к твоим ноутбукам и выдаёт выжимку — без переключения вкладок.
Установи расширение для браузера: найди «NotebookLM MCP» в Chrome Web Store
Войди в свой аккаунт Google в расширении
Добавь ноутбуки: скопируй ссылку на нужный ноутбук и вставь в расширение
Открой Claude Desktop → Customize → Connectors → нажми «+»
Выбери NotebookLM из списка доступных коннекторов
В чате напиши: «Дай мне топ-5 правил из моего ноутбука» → нажми «Разрешить 
всегда»
Совет: когда пишешь запрос, добавляй «воспользуйся моим ноутбуком» — и Claude сам 
пойдёт туда за информацией.
Требуется: Claude Desktop (в браузере не работает) + аккаунт Google
НАВЫК 5 ИЗ 12
Google Workspace — почта, календарь, файлы
Claude видит твоё расписание и почту. Добавляет события, читает письма, ищет файлы.
Открой Claude Desktop → Customize → Connectors
Нажми «Browse Connectors» и введи «Google»
Добавь Google Mail — нажми «+» → авторизуйся через браузер → выбери аккаунт → «Выбрать все» → Продолжить
Добавь Google Calendar — аналогично
Добавь Google Drive — аналогично
Открой новый чат и проверь: «Какие у меня события на завтра?»

черновики, искать по синтаксису
Google Calendar
смотреть события, 
создавать встречи, получать напоминания
Google Drive
искать файлы, читать

Совет: настрой ежедневный брифинг в Cawork — Claude будет сам присылать сводку по почте и задачам каждое утро.
Требуется: Claude Desktop + аккаунт Google
НАВЫК 6 ИЗ 12
Context7 MCP — актуальная документация
Claude всегда работает с последней версией библиотек. Никакого устаревшего кода.
Открой Claude Desktop → Developer → Edit Config
Открой файл claude_desktop_config.json в любом редакторе (например, VS Code или Cursor)
Добавь блок Context7 в раздел mcpServers
Сохрани файл
Перезапусти Claude Desktop
Проверь: напиши «Видишь ли ты Context7 MCP?» — Claude подтвердит
Код блока Context7:
{  "mcpServers": {    "context7": {      "command": "npx",      "args": ["-y", "@upstash/context7-mcp"]    }  }}    
Совет: при вайб-кодинге всегда добавляй в промпт «используй актуальную документацию через Context7» — и Claude сам подтянет нужную версию.
Требуется: Claude Desktop + Node.js. Полностью бесплатно.
НАВЫК 7 ИЗ 12
FireCrawl MCP — парсинг сайтов конкурентов
Claude вытаскивает любой сайт в Markdown. Идеально для анализа конкурентов и референсов.
Зарегистрируйся на firecrawl.dev (бесплатный план)
Пройди опросник и получи API-ключ в личном кабинете
Скопируй API-ключ (замазать на скриншоте!)
Открой Claude Desktop → новый чат
Вставь инструкцию установки из личного кабинета FireCrawl (там есть готовый 
промпт)
Claude сам установит MCP и подтвердит готовность
Совет: попроси Claude «обойди весь сайт example.com и верни все страницы в 
Markdown» — получишь полную структуру сайта конкурента за секунды.
Требуется: Claude Desktop + аккаунт на firecrawl.dev. Бесплатный план доступен.
НАВЫК 8 ИЗ 12
Notion MCP — агент ведёт задачи за тебя
Claude сам создаёт задачи, обновляет статусы и ищет файлы в твоём Notion.
Открой Claude Desktop → Customize → Connectors
Нажми «Browse Connectors» → введи «Notion»
Нажми «+» рядом с Notion → откроется страница авторизации
Войди в Notion → выбери, к каким страницам дать доступ → подтверди
Вернись в Claude — коннектор появится в списке активных
Проверь: «Найди в моём Notion задачи на эту неделю»
Создавать задачи и страницы Обновлять статусы (в работе → готово) Искать файлы и документы по запросу
Совет: попроси Claude вести документацию проекта прямо в Notion — он сам будет 
обновлять её после каждого изменения.
Требуется: Claude Desktop или браузер + аккаунт Notion
НАВЫК 9 ИЗ 12
Playwright MCP — Claude работает в браузере
Claude открывает браузер, кликает, заполняет формы и возвращает результат. Без твоего участия.
Открой Claude Desktop → Developer → Edit Config
Открой claude_desktop_config.json в редакторе
Добавь блок Playwright MCP в раздел mcpServers
Сохрани файл и перезапусти Claude Desktop
Проверь: «Работает ли Playwright MCP?»
Дай разрешение при первом запросе → «Разрешить всегда»
Код блока Playwright MCP:
{  "mcpServers": {    "playwright": {      "command": "npx",      "args": ["@playwright/mcp@latest"]    }  }}    
Режим без скриншотов
быстрый, структурированное взаимодействие со страницей
Визуальный режим
сохраняет скриншоты, точная навигация
Важно: работает только в Claude Desktop. В браузерной версии Claude не работает.
Требуется: Claude Desktop + Node.js
НАВЫК 10 ИЗ 12
Gamma MCP — презентации из диалога
Скажи Claude «сделай презентацию» — он найдёт твой шаблон и соберёт слайды сам.
Открой Claude Desktop → Customize → Connectors
Нажми «Browse Connectors» → введи «Gamma»
Нажми «+» → авторизуйся через аккаунт Gamma
Коннектор появится в списке активных
Открой новый чат и напиши: «Создай презентацию по этому диалогу»
Claude найдёт твой шаблон в Gamma и соберёт презентацию
Совет: заранее создай шаблон в Gamma со своим стилем — Claude будет использовать 
именно его. Так все презентации будут в едином стиле.
Требуется: Claude Desktop + аккаунт Gamma

НАВЫК 12 ИЗ 12
Skills Creator — создай любой навык за 10 минут
Claude сам исследует тему и превращает результат в готовый Skills. Без написания промптов вручную.
Открой Claude Desktop → новый чат
Нажми на иконку «Research» (глубокое исследование) в поле ввода
Напиши запрос: «Собери всё необходимое для создания Skills по [твоя тема]»
Дай Claude провести исследование (займёт 1–3 минуты)
Когда исследование готово, напиши: «Теперь создай Skills на основе этих данных»
Claude активирует встроенный Skills Creator и сгенерирует готовый навык → сохрани его в Customize → Skills
Совет: перед запросом улучши промпт через ChatGPT или Gemini (бесплатно) — это сделает исследование точнее и Skills качественнее.
Пример: «Собери всё необходимое для создания Skills по написанию высококонверсионных 
лендингов для SaaS» → Claude проведёт ресёрч → Skills Creator создаст навык.
Установка: встроено в Claude. Работает в Desktop и браузере.
