> [!abstract] ИИ Выжимка
> Статья описывает пошаговый процесс создания рабочего веб-сайта или продукта с нуля, используя комбинацию инструментов искусственного интеллекта. Сначала данные загружаются в NotebookLM для анализа и генерации структуры контента. Затем эта структура передается агенту Antigravity, который автоматически пишет чистый код (HTML/CSS/JS) на основе предоставленных данных. Завершающий этап — быстрая деплойментация готового сайта на внешних платформах, что позволяет превратить заметки в работающий продукт за минимальное время без навыков кодирования.

🛠 Частые проблемы и решения
Antigravity генерит на английском → Добавить в промпт: «Язык: русский» Продукт выглядит слишком ИИ-шно → Добавить стиль: «как Apple / Stripe / Linear» NotebookLM не берёт URL → Проверить что URL публичный, не за paywall Antigravity зависает Ctrl+Shift+P → Reload Window MCP не подключается → Запустить: nlm doctor Antigravity делает лишнее → Включить режим «Request Review» вместо «Always Proceed»
