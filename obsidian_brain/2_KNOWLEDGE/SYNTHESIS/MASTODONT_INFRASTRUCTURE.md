## 🎯 Суть Системы
Мастодонт — это не просто бот или набор скриптов, это **автономная бизнес-система**, построенная на принципе "Вечной Памяти" (Infinite Memory). Цель — превращение идей в MVP в режиме реального времени с сохранением 100% контекста.

## 🛠 Технологический Стек (Infinite Memory Stack)
1. **Claude Code / Antigravity**: "Руки" системы. Управление кодом, файлами и деплоем.
2. **NotebookLM + Obsidian**: "Мозг" и RAG. Синхронизация всех чатов для устранения технического долга.
3. **Gemma 4 + Ollama**: Локальный уровень приватности для обработки ПДН (Бобов, Зевс).
4. **Telegram MASTODONT Bot**: Интерфейс взаимодействия и воронка продаж.

## 📂 Объединенные темы из архивов
На основе анализа 35 проектов из ChatGPT:
### 1. Автоматизация и Скрипты
- **Sync Engine**: Использование `chatgpt_sync.py` для выгрузки истории.
- **Watcher**: Паттерн `ai_watcher.py` для автоматического сбора изменений в репозитории.
- **Summarizers**: Пакетная обработка через Ollama (Gemma 4).

### 2. Промпт-Инжиниринг (AI Protocol Stack)
- Использование структурированных промптов с ролями (Senior AI-Architect).
- Паттерн [Grounding]: Обязательная проверка `obsidian_brain` перед началом работы.

## 🚀 Вектор развития (2026 PRO)
- **Vibe Coding**: Переход от простого написания кода к агентскому управлению проектом.
- **Multiplication**: Автоматическое создание контента (видео, PDF-лидмагниты) на основе готового кода.
- **Privacy First**: Полная изоляция чувствительных данных в локальном контуре.
