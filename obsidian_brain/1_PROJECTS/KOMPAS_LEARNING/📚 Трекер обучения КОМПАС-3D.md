## 📝 Конспекты и практические заметки

### Тема 1. Компоновка сборки редуктора (Методология Top-Down)
> [!NOTE]
> **Суть метода:** Вместо моделирования отдельных деталей создается единая «Компоновочная геометрия» (КГ). Она служит «скелетом» и контейнером данных для сборки редуктора.
*   **Параметризация:** Параметры передачи (межосевое расстояние, модули, диаметры) привязываются к панели **«Переменные»**.
*   **Инструменты:**
    *   Создание вспомогательных плоскостей и локальных систем координат (ЛСК) для каждого вала (червячного и тихоходного).
    *   Построение эскизов контуров будущих колес.
    *   Использование функции **«Вставить функцию»** (v23/v24) в уравнениях переменных.
*   **Навык:** Формирование **«Коллекции геометрии»** для экспорта опорных контуров в файлы деталей без создания циклических ссылок.

### Тема 2. Сборка червячной передачи
*   **Метод:** Автоматическое построение колес и червяков на основе расчетных данных.
*   **Инструменты:** Приложение **«Валы и механические передачи 2D+3D»**. Ссылка по ЛСК через команду **«Копировать объекты»**.
*   **Навык:** Задание сопряжений вращения для симуляции кинематики червячной пары.

### Тема 3. Деталь «Корпус»
*   **Метод:** Моделирование сложного чугунного (или легкосплавного) корпуса редуктора на основе импортированных заготовок или литых прототипов.
*   **Инструменты:** Диагностика **«Проверка геометрии»** для выявления некорректного математического описания граней, прямое моделирование (**«Изменить положение грани»**, **«Удалить грани»**), зеркальные массивы крепежных ушек.
*   **Навык:** Редактирование импортированной CAD-геометрии без дерева истории построения.

### Тема 4. Деталь «Крышка»
*   **Метод:** Создание ответной части корпуса редуктора, расстановка 3D-размеров и шероховатости на электронной модели.
*   **Инструменты:** Простановка допусков формы и взаимного расположения поверхностей. Группировка конструктивных элементов по слоям (Ctrl+L).
*   **Навык:** Оформление трехмерных моделей деталей по ГОСТ 2.052-2015 (Электронная модель изделия).

### Тема 5. Сборка: Компоненты, Исполнения, Крепеж
*   **5.1 Локальные детали:** Создание мелких прокладок и колец прямо в сборке (хранятся без отдельного файла `.m3d`).
*   **5.2 Исполнения:** Настройка таблицы исполнений для изменения геометрического состояния уплотнений при затяжке.
*   **5.3 Стандартные изделия:** Вставка подшипников качения из справочника **Полином MDM** с преобразованием стандартного подшипника в заготовку для проточки под нестандартный размер.
*   **5.4–5.5 Крепежные группы:** Применение библиотеки **«Разъемные соединения»** для автогенерации пакетов болт-шайба-гайка по сетке точек компоновки.

### Тема 6. Состав изделия и спецификация (ЕСКД)
*   **Метод:** Автоматизированный выпуск спецификации на редуктор.
*   **Инструменты:** Панель **«Состав изделия»**, групповое управление свойствами (назначение марок материалов и масс), синхронизация форматов чертежей и спецификаций по ГОСТ 2.106-96.
*   **Навык:** Получение 100% корректной спецификации без ручного заполнения полей.

### Тема 7. Общие новинки базовой функциональности (v24)
*   **Направления:**
    *   Реверс-инжиниринг (совмещение сканированных STL-полигонов с твердотельными деталями по базовым точкам).
    *   Пакетное продление поверхностей.
    *   Копирование деталей через буфер обмена с сохранением сопряжений.

### Тема 8. Расчеты и Визуализация (v24)
*   **Инструменты:**
    *   **APM FEM:** Экспресс-анализ прочности валов и корпуса при пиковых крутящих моментах.
    *   **KOMPAS Flow:** Гидродинамическое моделирование циркуляции масла в полости редуктора.
    *   Модуль встроенного **рендеринга** для подготовки презентационных материалов редуктора.
