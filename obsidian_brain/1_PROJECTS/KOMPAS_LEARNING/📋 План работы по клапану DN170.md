# 📋 Адаптированный план работы: Клапан DN170

> **База знаний:** NotebookLM — [МИФИ НГТИ](https://notebooklm.google.com/notebook/7d4e0e25-2e5e-4a66-8e9b-6e7422333622) + [КОМПАС-3D курс](https://notebooklm.google.com/notebook/22bb2c90-d304-42d8-992f-1da1a7e57721)
> **Нормативная база:** ТР ТС 032/2013, ГОСТ 34347, НП-016-05, НП-089-15

---

## 🎯 Цель проекта
Спроектировать клапан DN170 для атомного производства с массой **≤ 22,15 кг**, обеспечив соответствие ядерным стандартам безопасности.

---

|---|--------|----------------------|--------|
| 1.1 | Изучить требования к сосудам под давлением | **ТР ТС 032/2013** | ⬜ |
| 1.2 | Изучить требования к оборудованию АЭС | **НП-016-05, НП-089-15** | ⬜ |
| 1.3 | Проверить требования к сварным швам | **ГОСТ 34347** | ⬜ |
| 1.4 | Определить класс безопасности арматуры | Внутренний регламент ЦПТИ | ⬜ |
| 1.5 | Получить номер проекта в архиве | Регламент №311/01-01/72-Р-АХД | ⬜ |

## ⚙️ Нормативная шпаргалка

### Ключевые стандарты
```
ТР ТС 032/2013  → Оборудование под давлением (главный TR)
ГОСТ 34347      → Стальные сварные сосуды (расчёт стенок)
НП-016-05       → Ядерная безопасность: классификация
НП-089-15       → Правила устройства и безопасности АЭУ
ГОСТ 2.106-96   → ЕСКД: Спецификация
```

### Кодировка ЦПТИ
```
Формат: РПНТ-XXX-XX-XX-XX-XXX-XXX
Код 20  → Газы
Код 22  → Азот  
Код 50  → Криогенная система
```

> ⚠️ Если в исходниках не указана марка стали или давление — статус **«ДАННЫЕ ОТСУТСТВУЮТ»**, запросить уточнение!
