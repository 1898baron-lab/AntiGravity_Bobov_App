# 🛠️ Деталировка мангала #812
*Толщина металла: 4 мм (рекомендуемая)*

## 1. Печь под казан (Цилиндр + Основание)
- **Обечайка (развертка):** 1200 x 350 мм (при Ø380 мм).
- **Дно печи (круг):** Ø380 мм.
- **Дверца:** 150 x 200 мм (вырезается из обечайки).

## 2. Жаровня (Прямоугольный ящик)
- **Дно жаровни:** 600 x 350 мм.
- **Боковина длинная (2 шт):** 600 x 180 мм (с вырезами под шампуры).
- **Боковина короткая (1 шт):** 350 x 180 мм.
*Примечание: Четвертая сторона — это стенка печи.*

## 3. Ножки
- **Труба профильная 25x25:** 4 шт. по 700 мм.

## 4. Спецификация раскроя (Laser Cut)
| Деталь | Кол-во | Размер (мм) | Площадь (м2) |
| :--- | :--- | :--- | :--- |
| Развертка печи | 1 | 1200 x 350 | 0.42 |
| Дно печи | 1 | Ø380 | 0.11 |
| Дно жаровни | 1 | 600 x 350 | 0.21 |
| Стенка длинная | 2 | 600 x 180 | 0.22 |
| Стенка торцевая | 1 | 350 x 180 | 0.06 |
| **ИТОГО** | | | **~1.02 м2** |

## 💡 Советы по сборке
1. Сначала сваривается цилиндр печи.
2. Затем к нему приваривается прямоугольный ящик жаровни.
3. В последнюю очередь привариваются ножки с усилением (косынками).
