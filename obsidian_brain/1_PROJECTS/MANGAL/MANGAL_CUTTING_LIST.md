# Спецификация для лазерной резки (Мангал "MASTODONT")

**Материал:** Сталь горячекатаная (Г/К) 4 мм.
**Общий вес (примерный):** 18.5 кг.

## Детали
| Наименование | Кол-во | Размеры (мм) | Примечание |
| :--- | :--- | :--- | :--- |
| Дно | 1 | 600 x 300 | С отверстиями для поддува (диаметр 10мм) |
| Стенка длинная (фасад) | 2 | 600 x 200 | С логотипом "MASTODONT" (прорезь) |
| Стенка короткая (боковина) | 2 | 300 x 200 | С пазами для сборки "шип-паз" |
| Ножки (усиленные) | 4 | 500 x 60 | Г-образный профиль (из двух частей) |
| Зольник (выдвижной) | 1 | 580 x 280 | Опционально (сталь 2 мм) |

## Инструкции для производства
1. **Резка:** Использовать азот для чистого реза без окалины.
2. **Логотип:** Прорезать только на передней стенке.
3. **Пазы:** Допуск на сборку +0.2 мм к толщине металла (итого 4.2 мм паз).
4. **Отверстия:** Расстояние между центрами отверстий на дне — 50 мм.
