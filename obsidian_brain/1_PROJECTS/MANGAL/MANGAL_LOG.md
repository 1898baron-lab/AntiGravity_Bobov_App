# 🔥 Проект: Мангал (Рабочая Документация)
*Заказ: Разработка РД на основе фото-референса*

## 🎯 Цель
Выпустить полный комплект КД (Конструкторской Документации) для изготовления:
- Сборочный чертеж (СБ).
- Чертежи деталей (для лазерной резки).
- Спецификация.

## 🗺️ План действий
1. [ ] **Анализ референса:** Оценить габариты по фото и пожеланиям заказчика.
2. [ ] **3D Моделирование:** Создать параметрическую модель в КОМПАС-3D (с учетом толщины металла, например 3-5 мм).
3. [ ] **Развертки:** Получить DXF файлы для лазерной резки.
4. [ ] **Оформление:** Выпустить чертежи по ГОСТ.
