# Summary: Инженерная память и ИИ
**Оригинал:** [[Инженерная память и ИИ]]

# Краткая суть
Успешно настроена асинхронная коллаборационная экосистема для проектирования, где Obsidian выступает как основная база знаний, синхронизированная через GitHub с внешним AI (ChatGPT). Основной фокус — разработка Технического Задания (ТЗ) на пилотный стапель 1200x800, ожидая струкгурирующий ответ от ChatGPT для перехода к расчетам WPS.

# 📌 Ключевые факты и параметры
*   **Проект:** Пилотный стапель (Stapel) 1200x800.
*   **База знаний (KB):** Obsidian (`obsidian_brain/Engineering/Machine_Building_Tech.md`).
    *   Содержит информацию по стандартам ЦПТИ и требованиям Росатома.
    *   Доступ к поиску реализован через инструменты `search_knowledge` и `fetch_document`.
*   **Workflow:** Асинхронный обмен данными: `TO_CHATGPT.md` (входной контекст) $\to$ ChatGPT $\to$ `FROM_CHATGPT.md` (ответ).
*   **Инфраструктура AI Bridge:**
    *   Скрипт: `mcp_chatgpt_bridge.py`.
    *   Используемый порт: 8000.
    *   Обнаружена проблема: Конфликт адресов сокета (Winerror 10048) при запуске сервера.
*   **Системный дашборд (MASTADONT):**
    *   Интеграции: Claude Connector (localhost:8765), NotebookLM MCP, Obsidian Brain, ChatGPT Bridge (STANDBY).
    *   Репозиторий синхронизации: `https://github.com/1898baron-lab/AntiGravity_Bobov_App`.
*   **Документация:** Начинается формирование черновика ТЗ в файле `obsidian_brain/Engineering/Pilot_Stapel_1200x800_TZ.md`.
