# Проект: Систематизация знаний и разработка инженерных/личностных промтов

## 📋 Общее описание
Это комплексная сессия по извлечению и структурированию знаний из различных источников: от высокоспециализированного инженерного проектирования (ЕСКД, технические карты оснастки) до бизнес-стратегии личного преимущества и коммуникационных моделей. Основная ценность — разработка и проверка шаблонов (промтов) для генеративных систем, способных работать в режиме экспертной симуляции, что позволяет автоматизировать процесс технического контроля и генерации документации.

## 📌 Ключевая фактура
**Технические и методологические параметры:**
*   **Проектирование оснастки:** Требуемая техническая карта должна включать: конструкцию (привязка, зажимы), маршрут операций (сварка, механическая обработка, термообработка, покраска), спецификацию материалов (марка стали, фурнитура), контрольные размеры и допуски.
*   **Технический контроль (КД):** Проверка должна быть структурирована по чек-листу и обязательно ссылаться на нормативные документы (ГОСТ, ЕСКД, ЕСТД) для выявления несоответствий (именование, масштаб, допуски, клейма сварки, шрифты, спецификации).
*   **Нормативная база:** Критически важны ГОСТы, ЕСКД (Единая система конструкторской документации) и ЕСТД (Единая система терминологической документации).

**Стратегические концепции и инсайты:**
*   **Unfair Advantage (Нечестное преимущество):** Любое устойчивое конкурентное преимущество (Moat), которое должно быть системно воспроизведено. Источники такого преимущества: знания, связи, репутация, доступ к каналам (Модель MILES: Money, Intelligence, Location, Education, Status).
*   **Моделирование коммуникации:** Разработаны устойчивые стратегии общения со старшим поколением: **«Спокойно $\rightarrow$ По делу $\rightarrow$ С уважением»**. Цель — не победа в споре, а сохранение отношений и ясность передачи информации.
*   **AI Tools:** Для генерации видеоконтента (пп. «Праздничное видео») используется инструмент Sora. Важна информация о географической доступности (в данный момент поддерживаются США, Канада, Япония и некоторые страны Латинской Америки).

**Контактные и личные данные (Профиль):**
*   **Квалификация:** Инженер-конструктор 2-й категории (АО «ЦПТИ», Росатом).
*   **Экспертиза:** Проектирование производственной оснастки, создание цифровых двойников участков, разработка конструкторской документации по ЕСКД.
*   **Сферы интересов/хобби (для профиля):** Судостроение, Проектирование, CAD/CAM, SolidWorks, Компас-3D, HVAC/ОВиК, Парусный спорт, ИИ в проектировании, Металлообработка.

## 🚀 Action Items & Статус
- [ ] **Протестировать и откалибровать инженерные промты:** Использовать готовые шаблоны для генерации полного комплекта документации (КД, Тех. карты) на реальные, конкретные объекты, указав номера РД и габариты.
- [ ] **Структурировать модель Unfair Advantage:** Превратить теоретические тезисы в личный чек-лист привычек для ежедневного отслеживания и повышения личной/проектной системности.
- [ ] **Выбрать и адаптировать «Позицию»:** Финализировать личную позицию для презентаций/подкастов, используя формулу «Принятие $\rightarrow$ Уверенность $\rightarrow$ Действие» для публичного позиционирования.
