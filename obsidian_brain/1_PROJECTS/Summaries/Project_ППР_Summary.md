# Проект: Разработка и стандартизация ППР для систем ОВК и коммуникаций АЭС

## 📋 Общее описание
Данный проект сфокусирован на разработке, систематизации и технической проверке Проектов Производства Работ (ППР) для критически важных инженерных систем (ОВК, ХС, отопление) на объекте АЭС. Обсуждения охватывают как глубокий технический анализ рабочей документации (РД) — включая проверку замечаний Заказчика и расчёт логистики (автокраны, подъёмники), — так и методологическое обеспечение процесса. Ключевым элементом является использование продвинутых возможностей AI-помощников (ChatGPT Projects) для сохранения контекста, стандартизации шаблонов и повышения эффективности работы инженеров ПТО.

## 📌 Ключевая фактура

**Технические параметры и нормативы:**
*   **Объект:** Атомная электростанция (АЭС).
*   **Системы:** ОВК (Отопление, Вентиляция, Кондиционирование), ХС (Холодоснабжение), трубопроводы СПВ 45/70.
*   **Документация:** Рабочая документация (РД) по разделу АР (Архитектурные решения) и инженерным разделам (HV, TM, KM и др.).
*   **Методология:** Требуется полная разработка Технологических карт (ТК) на монтаж сварных соединений, ремонт временных опор, и на монтаж трубопроводов в осях A-B/7-10 с помощью подъемника.
*   **Эксплуатационное требование:** Учёт габаритов и маршрутов перемещения крупногабаритного оборудования (вентмашблоки, чиллеры) по кровле (Габариты кровли ≈ 60 000 × 18 000 мм).

**Важные контакты и роли:**

*   **Внутренние стейкхолдеры:** Бобов Кирилл (Основной исполнитель/инициатор), Смирнова Евгения (Ответственная за ТК).
*   **Специалисты:** Валюк Дмитрий Геннадьевич (Утверждение замечаний), Артем (Помощь в анализе РД).
*   **Компания:** ООО «ИнСетКом» (Рабочая структура).

**Уникальные идеи и процессы (Knowledge Artifacts):**
1.  **AI-Проектная методика:** Использование функции Projects в ChatGPT для создания изолированного, контекстно-обогащённого рабочего пространства, объединяющего чаты, файлы и инструкции.
2.  **Идеальная структура папок для ППР-инженера:**
    *   `/01_RD` (Чертежи)
    *   `/02_PPR` (Шаблоны, версии, графика)
    *   `/03_TI_KMD` (Инструкции, КМД)
    *   `/04_Reviews` (Замечания Заказчика, Excel-трекеры)
    *   `/05_Analytics` (Отчеты, расчёты)
    *   `/ZZ_Archive` (Архивные/устаревшие версии)
3.  **Чек-лист анализа АР (АР-Аналитика):** Систематизированный процесс извлечения данных из АР:
    *   Основные оси, габариты.
    *   Типы проемов (двери, ворота, люки, шахты).
    *   Возможные маршруты перемещения (выявление "узких мест").
    *   Точки крепления (закладные детали, конструкция).
4.  **Топ-5 AI-запросов для ППР (Методологическая база):**
    *   Анализ РД и генерация карты правок (с цитированием СП/ГОСТ).
    *   Формирование полного шаблона Технологической карты (ТК).
    *   Проверка полноты нормативной базы (СП, ГОСТ, СТО) с указанием сроков обновления.
    *   Расчет базовых трудозатрат и логистики (ФОТ, механизмы).
    *   Создание универсального чек-листа КК/ОТ по этапам работ.

## 🚀 Action Items & Статус

*   [ ] **Проектирование:** Финализировать расчёт размещения автокранов, основываясь на уточнённых данных о загрузке оборудования и несущей способности грунта.
*   [ ] **Методология:** Интегрировать разработанный Чек-лист АР и ППР в основную инструкцию проекта для стандартизации работы всех инженеров ПТО.
