# Проект: Цикл проектирования в КОМПАС-3D по ЕСКД (Mini-прижим)

## 📋 Общее описание
Данный проект представляет собой комплексную дорожную карту освоения полного цикла машиностроительного проектирования, используя программу КОМПАС-3D. Основная цель — создание функционального изделия («Мини-прижим для сверления») и полный комплект технической документации (сборка, детали, спецификация) за 14 дней. Весь процесс строго регламентирован и опирается на актуальные стандарты ЕСКД.

## 📌 Ключевая фактура
**Технические параметры и стандарты:**
*   **Процесс:** 3 детали $\rightarrow$ 1 сборка $\rightarrow$ 1 спецификация.
*   **Геометрия/Материалы (Пример):**
    *   Базовая плита: 160×100×12 (Ст3). Отверстия: 2×⌀8H7, 4×М6-6H.
    *   Опора/Призма: 40×40×20 (Ст3). V-образная выборка 90°.
    *   Прижимная планка: 80×20×10 (45). Паз 8×20, отверстие М6-6H.
*   **Стандарты (ГОСТ):** В рамках обучения проработаны:
    *   ГОСТ 2.301, 2.303, 2.304, 2.302 (Форматы, линии, шрифт, масштабы).
    *   ГОСТ 2.104 (Шаблон листа, основная надпись, формат 1 по ГОСТ 2.104).
    *   ГОСТ 2.307, 2.308, 2.309 (Размеры, допуски формы/позиции, шероховатость Ra 3,2/6,3).
    *   ГОСТ 2.701–2.704 (Обозначения в узлах, сборок и частей).
    *   ГОСТ 2.108 (Структура и заполнение спецификации).
*   **Артефакты:** Шаблон листа А3, штамп с переменными полями, сборка, спецификация.

**Важные параметры процесса:**
*   **Методология:** Использование базы измерений (базы A/B/C) для определения позиционных допусков.
*   **Структура спецификации:** Документация / Сборочные единицы / Детали / Стандартные изделия / Прочие / Материалы / Комплекты.
*   **Контрольные точки (КТ):** Готовность шаблона (D1), готовность 3 деталей с допусками (D7), готовность сборки (D10), финальный PDF-пакет (D13).

**Контакты/Ссылки:**
