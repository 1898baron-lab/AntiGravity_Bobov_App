**Оригинал:** [[Приветствие беседы]]

# Краткая суть
Диагностика проблемы с использованием функции голосового ввода (Speech Recognition) в операционной системе Windows 10. Основная сложность заключается в том, что стандартные встроенные средства ОС, по мнению пользователя, не поддерживают ввод текста на русском языке, предоставляя возможность только для английского.

# 📌 Ключевые факты и параметры
- **ОС:** Windows 10.
- **Функция:** Голосовой ввод (Speech Recognition).
- **Проблема:** Недоступность функции на русском языке, ограничение функционала только английским.
- **Встроенные средства:** Упомянута функция Microsoft Dictate (как потенциальное решение).
- **Рекомендации по решению:** Использование сторонних программных комплексов для обеспечения поддержки голосового ввода на русском языке.
