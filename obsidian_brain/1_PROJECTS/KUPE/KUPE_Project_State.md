# КУПЕ — Пневматическая установка дистанционного сверления

## Описание
Пневматическая установка дистанционного сверления заглушек контейнеров ОГФУ.

## Ключевые узлы
- Зажимной узел: 4×Ø80мм цилиндры SMC
- Захват выдвижной: Сталь 20, HRC 40-45
- Схема: бистабильный распределитель
