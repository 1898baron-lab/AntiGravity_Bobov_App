# Схема базирования фланца Ø280 (ГОСТ 21495-76)

Для обеспечения точности IT7 при токарной обработке используется схема "3+2+1" (Шесть точек).

## 🔲 ASCII-Визуализация

```text
       Z+ (Ось вращения)
       ^
       |          [Заготовка Фланец]
       |       _______________________
       |      |           |           |
 [3] --|-->   |   (4)     |    (5)    |  <-- [Кулачки патрона]
       |      |___________|___________|
       |                  |
       |        [1]      [2]      [3]    <-- [Установочные опоры на торце]
       |__________________________________
       |__________________________________  [Планшайба/Патрон]
       |
```

## 📍 Описание опорных точек:
1. **Точки 1, 2, 3 (Установочная база)**: Лишают деталь 3-х степеней свободы (перемещение по Z и вращение вокруг X, Y). Располагаются на торце фланца.
2. **Точки 4, 5 (Направляющая база)**: Лишают деталь 2-х степеней свободы (перемещение по X, Y). Реализуются через самоцентрирующиеся кулачки патрона.
3. **Точка 6 (Упорная база)**: В данной операции ограничивает вращение относительно оси Z (силы трения от зажима).

---
## 🏁 Технологические рекомендации:
- **Базирование**: Исходной базой является литой/черновой торец фланца.
- **Точность**: Для IT7 биение торца относительно оси не должно превышать 0.05 мм.
- **Зажим**: Усилие зажима кулачков должно быть распределено равномерно во избежание деформации облегченного фланца (толщина 22 мм!).
