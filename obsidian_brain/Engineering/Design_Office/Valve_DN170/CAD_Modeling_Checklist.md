# Чек-лист: 3D-моделирование клапана 10.129 (Версия АМг6)

Этот список — твоя инструкция по обучению в КОМПАС-3D. Цель: **21.43 кг**.

## 🛠 1. Настройка свойств (Первым делом!)
- [ ] **Материал**: Всем деталям (Корпус, Фланцы, Тарелка) назначить **"Алюминиевый сплав АМг6 ГОСТ 4784-97"**.
- [ ] **Плотность**: Проверить значение $\rho \approx 2650 \, \text{кг/м}^3$ в свойствах.
- [ ] **Цвет**: Назначь деталям разные цвета (например, корпус — светло-серый, вал — темно-серый), чтобы лучше видеть сборку.

## 📐 2. Геометрия (Ключевые изменения)
- [ ] **Корпус**: Толщина стенки обечайки — **5.0 мм**.
- [ ] **Фланцы**: Внешний диаметр **Ø280**. Толщина чистовая — **10 мм** (вместо 24).
- [ ] **Проходное сечение**: Внутренний диаметр — **Ø172 мм**.
- [ ] **Резьбы**: В Компасе используй инструмент "Условное обозначение резьбы" для M200x2.

## ⚖️ 3. Весовой контроль
- [ ] Выполнить команду **"МЦХ модели"**.
- [ ] Твоя цель: **21.43 кг** (в сборе с приводом 5 кг).
- [ ] Если вес меньше — отлично, у нас есть запас на крепеж.

## 🏁 4. Сборка и Анализ
- [ ] **Сопряжения**: Используй "Соосность" для фланцев и вала, "Совпадение объектов" для торцов.
- [ ] **Пересечения**: Проверь "Проверку коллизий". Тарелка не должна врезаться в корпус при открытии.
