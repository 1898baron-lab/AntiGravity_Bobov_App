3. Полуавтоматизация через экспорт и скрипт

Экспорт

Bash
openai api chat.export --conversation-id <ID_Анализ_РД> --format json > 00USV_rd.json
openai api chat.export --conversation-id <ID_Контекст>   --format json > 00USV_ctx.json
openai api chat.export --conversation-id <ID_Изображения> --format json > 00USV_img.json

Скрипт-объединитель (Python + OpenAI API):

Python
Запустить
import json, glob, openai

openai.api_key = "твой_ключ"

def load_and_concat(files):
    text = ""
    for f in files:
        data = json.load(open(f, encoding="utf-8"))
        # Собираем только ответы assistant
        msgs = [m["content"] for m in data["messages"] if m["role"]=="assistant"]
        text += "\n\n".join(msgs) + "\n\n"
    return text

chats = ["00USV_rd.json","00USV_ctx.json","00USV_img.json"]
full_text = load_and_concat(chats)

prompt = (
    "Ты — эксперт-инженер ППР проекта «00USV».\n"
    "На основе всех ответов из трёх чатов собери сводный документ по разделам ППР:\n"
    "1. Общие данные\n"
    "2. Сведения о проекте-основании\n"
    "3. Календарный план\n"
    "4. Технологические карты\n"
    "5. Качество и контроль\n"
    "6. Охрана труда и охрана природы\n"
    "В каждом разделе дай только релевантные тезисы."
)

resp = openai.ChatCompletion.create(
    model="gpt-4o-mini",
    messages=[
        {"role":"system","content":prompt},
        {"role":"user","content": full_text}
    ]
)

print(resp.choices[0].message.content)

Результат — готовый консолидационный документ.
