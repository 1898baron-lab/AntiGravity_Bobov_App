Привет! Есть два способа — зависит от того, что тебе нужно: обозначение или настоящая геометрия.

Способ 1 — Условное обозначение (99% случаев для КД)

Это стандартный путь — геометрия резьбы не строится, только обозначение для чертежа.

Выбери торец или цилиндрическую поверхность стержня
Операции → Элементы тела → Условное изображение резьбы
Укажи поверхность цилиндра
Задай параметры: тип (метрическая, дюймовая и т.д.), шаг, длину
ОК — появятся тонкие линии резьбы

На чертеже отобразится по ГОСТ 2.311-68 автоматически.

Способ 2 — Реальная геометрия (для визуализации / МКЭ)
Построй спираль: Операции → Пространственные кривые → Спираль цилиндрическая
Шаг = шаг резьбы
Число витков = длина / шаг
Нарисуй профиль резьбы (треугольник 60° для метрической) в плоскости, перпендикулярной спирали
Операции → Элементы тела → Кинематическая операция — профиль по спирали
Вычти результат из цилиндра булевой операцией (Вычитание)

⚠️ Реальная геометрия резьбы резко тормозит модель и сборку — используй только когда реально нужно.

Для резьбового отверстия (внутренняя резьба) — то же самое, только выбираешь внутреннюю поверхность.
