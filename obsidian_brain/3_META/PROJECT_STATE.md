# ANTIGRAVITY PROJECT STATE
*Последнее обновление: 2026-05-19*

## 1. Current Status (Текущий статус)
Проект реорганизован согласно новым приоритетам пользователя:
1. **Учебный Хаб (Приоритет №1):** Обучение по КОМПАС-3D v24, практическая стажировка по разработке быстродействующего ядерного клапана Ду170 (проект «ЗЕВС») под руководством Афанасьева А.И. и инженерная нормативно-расчетная база знаний объединены в единый высокоприоритетный трек. Проект сборочно-сварочного стапеля **«ЗАКРЕП v1.0»** успешно выведен из разработки и перенесен в архив (`4_ARCHIVE/ZAKREP/`).
2. **Юридический блок (Приоритет №2):** Подготовлена детальная жалоба по ст. 125 УПК РФ в интересах Олега Бобова.

## 2. Tech Stack (Стек технологий)
*   **ИИ Движок:** Локальная модель Ollama (`gemma4:26b-lite` / `gemma4:26b`)
*   **САПР Автоматизация:** КОМПАС-3D v24 API (`win32com.client` на Python)
*   **База Знаний:** Obsidian (папка `obsidian_brain`)
*   **Тестирование:** pytest (4/4 тестов ПРОХОДЯТ ✅)
