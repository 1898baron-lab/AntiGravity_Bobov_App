## 🟣 Obsidian — Что нового (v1.12, февраль–март 2026)

### 🔥 Крупные новые фичи:

**1. Obsidian CLI** — управление из терминала
```bash
obsidian open "MyNote.md"
obsidian search "Бобов"
obsidian daily:path
obsidian rename "OldNote.md" "NewNote.md"
```
> Важно для нас: можно вызывать из Python-скриптов напрямую вместо записи файлов вручную

**2. Bases** — встроенная база данных (как Notion/Airtable)
- Drag & drop импорт файлов в таблицу
- Поиск и фильтрация
- Контекстное меню на строках
- Потенциально заменяет плагин Advanced Tables для опisi имущества Бобова

**3. Resizable Images** (Live Preview)
- Перетащи угол картинки — изменится размер
- Двойной клик — сброс

**4. Автоочистка вложений**
- При удалении файла — предложит удалить его вложения

**5. iOS Share Extension**
- Сохранять веб-страницы в vault прямо из браузера

**6. Obsidian Headless Sync**
- Синхронизация vault без GUI, через npm `obsidian-headless`
- Годится для серверного бэкапа

### ✅ Полезные мелочи:
- Вставка URL при выделении текста → `[текст](url)` автоматически
- Копирование текста = HTML (удобно для Google Docs)
- Canvas backlinks видны в Graph View
- Ctrl+C/V в File Explorer — копирование файлов
- Fuzzy поиск стал умнее при пробелах в запросе
