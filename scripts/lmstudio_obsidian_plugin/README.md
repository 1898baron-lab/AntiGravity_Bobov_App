# AntiGravity RAG Plugin для LM Studio

Этот плагин позволяет моделям внутри LM Studio (например, Llama 3, Qwen) напрямую обращаться к локальной базе знаний Obsidian (`C:\ANTIGRAVITY\1\obsidian_brain`).

## Установка и запуск

1. Перейдите в эту директорию:
```bash
cd C:\ANTIGRAVITY\1\scripts\lmstudio_obsidian_plugin
```

2. Установите зависимости (если еще не установлены):
```bash
npm install
```

3. Запустите плагин в режиме разработки (для этого должен быть установлен CLI `lms` от LM Studio):
```bash
npm run dev
```

## Как использовать
После запуска плагина, откройте десктопное приложение **LM Studio**.
В чате с любой локальной моделью появится возможность использовать инструменты (Tools). 
Спросите модель:
> "Найди в Obsidian информацию по слову ЕСКД"
или
> "Прочитай досье ZEUS и скажи, какой зазор на тепловое расширение"

Модель сама вызовет плагин, прочитает файлы с вашего жесткого диска и даст ответ, сохраняя 100% приватность!
