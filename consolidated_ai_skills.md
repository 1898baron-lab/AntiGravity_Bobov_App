## 🚀 SECTION 1: The "Infinite Memory" Stack
*Как перестать терять контекст и построить систему, которая умнеет с каждым днем.*

### 🛠 Tools: NotebookLM + Obsidian + Claude
- **NotebookLM as RAG:** Используйте его не для чтения, а как внешнюю базу данных (Grounding). Закидывайте сюда транскрипты, PDF, документацию.
- **Obsidian (The Local Brain):** Храните знания в `.md`. Это лучший формат для Claude Code и Cowork.
- **Связка "AntiGravity + NotebookLM":** Позволяет за 20 минут превращать заметки в работающие сайты/лендинги.
    - *Алгоритм:* Сбор данных в Notebook -> Исследование (Research) -> Перенос структуры в AntiGravity -> Генерация кода.

### 🧠 Persistent Brain (Claude Projects)
- **Problem:** "Amnesia" (потеря контекста в чатах).
- **Solution:** Claude Projects.
    - **Instructions:** Залочьте свой "Brand Voice", правила и тон.

    2. Описание логики ("Вайб"). 
    3. Контроль выполнения агентом.
