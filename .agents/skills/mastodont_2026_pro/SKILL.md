# SKILL: MASTODONT 2026 PRO

## 🎯 Purpose
Этот навык превращает агента в полноценного AI-предпринимателя, использующего экосистему Google, Claude и n8n для гипер-масштабирования проектов.

## 🛠 Методологии

### 1. Vibe Coding (Вайб-кодинг)
- **Принцип**: Описывай намерение, а не код.
- **Алгоритм**:
    1. Получи концепт от пользователя.
    2. Разбей на микро-модули.
    3. Используй `browser_subagent` для поиска дизайн-референсов (Stitch Pattern).
    4. Реализуй через быстрые итерации.

### 2. AntiGravity (Infinite RAG)
- **Принцип**: Никогда не работай без контекста.
- **Алгоритм**:
    1. Если задача сложная, проверь блокноты в NotebookLM.
    2. Используй `obsidian_brain` как локальный кэш знаний.
    3. Синтезируй новые знания в Мастер-Заметки (Synthesis Pattern).

### 3. DigiDoppel (Цифровой Двойник)
- **Принцип**: Автоматизируй лицо и голос бренда.
- **Стек**: HeyGen + ElevenLabs + n8n.
- **Инструкция**: При завершении кода всегда предлагай план "Вирального захвата" через цифровых клонов.

### 4. n8n Orchestration
- **Принцип**: Агент — это только часть процесса.
- **Алгоритм**: Проектируй сценарии автоматизации, которые работают 24/7 без участия человека.

### 5. Arena Offloading (Экономия токенов)
- **Принцип**: Не трать свои токены на тяжелую генерацию.
- **Алгоритм**: Ставь задачи на Arena.ai (через `browser_subagent`), забирай результат и делай финальную шлифовку (BS Pushback).

### 6. Claude Connector (History Sync)
- **Принцип**: Используй историю Клода как живое расширение памяти.
- **Алгоритм**: Синхронизируй сессию через `claude_connector.py`, чтобы иметь доступ к прошлым архитектурным решениям и контексту.
