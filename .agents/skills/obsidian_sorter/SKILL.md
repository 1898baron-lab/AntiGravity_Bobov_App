# Obsidian Sorter (Авто-сортировщик)

Этот навык позволяет автоматически обрабатывать "сырые" заметки и статьи, сохраненные с помощью Web Clipper.
Он читает файлы из папки `Clippings`, отправляет их локальному ИИ (например, LM Studio) для выделения главных тезисов, назначения релевантных тегов и категорий. Затем он обновляет файл, добавляя красивый заголовок и YAML-фронтматтер, и перемещает его в структуру `2_KNOWLEDGE/WIKI`.

## Структура
- `scripts/sorter.py`: Основной скрипт для запуска сортировщика.

## Использование
Скрипт можно запускать вручную или через планировщик задач (cron/schtasks) для регулярной уборки в базе знаний.

```powershell
python .agents/skills/obsidian_sorter/scripts/sorter.py
```
