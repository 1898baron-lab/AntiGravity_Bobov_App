# ChatGPT Archiver Skill

Этот навык предназначен для глубокой синхронизации содержимого ChatGPT (проекты, чаты) с локальным архивом.

## Возможности
- Подключение к запущенному экземпляру Chrome через порт 9222.
- Автоматическое раскрытие списка проектов (нажатие кнопки "Больше").
- Прокрутка бокового меню для обнаружения всех доступных элементов.
- Скачивание сообщений из чатов в формате Markdown.
- Сохранение оглавления для каждого проекта.

## Требования
1. Запущенный Chrome с флагом удаленной отладки:
   `chrome.exe --remote-debugging-port=9222`
2. Установленные зависимости: `playwright`.
3. Виртуальное окружение проекта (`.venv`).

## Использование

### 1. Синхронизация всех проектов
Запустите основной скрипт:
```bash
.venv\Scripts\python.exe .agents\skills\chatgpt_archiver\scripts\chatgpt_sync.py
```

### 2. Структура вывода
Файлы сохраняются в:
`C:\ANTIGRAVITY\1\obsidian_brain\Archives\ChatGPT_Projects\`

## Правила на будущее (Best Practices)
1. **CDP Connection**: Всегда использовать подключение по порту 9222, чтобы избежать проблем с авторизацией (Cloudflare/Login).
2. **Dynamic UI**: Использовать JS-инъекции (evaluate) для поиска кнопок по тексту и координатам, так как селекторы ChatGPT часто меняются.
3. **Deduplication**: Проверять существующие файлы по размеру перед скачиванием, чтобы экономить время.
4. **Encoding**: Всегда принудительно устанавливать кодировку UTF-8 в Python-скриптах для корректной работы в Windows-консоли.
