# 🛠️ Скилл: Инженер-Генератор (Arena.ai Designer)

## Описание
Этот скилл предназначен для работы с платформой **Arena.ai** (в частности, моделями `kimi-k2-thinking-turbo` и `tetra-0429-5`). Он помогает автоматически генерировать спецификации (BOM), технические требования и текстовые описания чертежей для инженерных проектов (например, Клапан DN170, Модульный мангал).

## 🚀 Возможности моделей Arena.ai
- **Что они могут:**
  - Генерировать детализированные технические спецификации (ТЗ).
  - Создавать Bill of Materials (BOM) — ведомости материалов.
  - Составлять отчеты с расчетами (на уровне текста).
  - Выдавать текстовые спецификации для чертежей (координаты, размеры, описания для переноса в CAD).
- **Ограничения:**
  - Модели утверждают, что **не могут генерировать SVG/картинки напрямую в текстовом режиме**.

## 📝 Шаблон Промпта для Arena.ai (Ручной режим)

Используйте этот шаблон при работе в браузере или через API:

```text
Роль: Ведущий инженер-конструктор.
Задача: Разработать техническую спецификацию и BOM (Bill of Materials) для проекта [НАЗВАНИЕ ПРОЕКТА, например, "Модульный мангал" или "Клапан DN170"].

Требования к ответу:
1. Выдать подробный перечень всех необходимых деталей с указанием материалов (по ГОСТ/НТД) и их количества.
2. Подготовить текстовое описание габаритов и допусков для создания 3D-модели.
3. Указать ключевые требования к производству (например, сварные швы, обработка).
4. Оформить результат в виде структурированных таблиц Markdown.
```
