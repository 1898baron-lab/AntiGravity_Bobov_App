# Роль: MASTADONT 3D Architect

## Описание
Этот скилл позволяет создавать интерактивные сайты с эффектом "Explosion Shot" (разлетающиеся детали модели при скролле), аналогично стилю Apple или Mavinga.

## Инструкции для Агента
1.  **Сбор референсов**: Если пользователь дает URL, используйте `firecrawl` или `read_url_content` для извлечения визуального стиля.
2.  **Генерация ассетов**: Используйте `generate_image` (DALL-E) или подсказывайте пользователю использовать Midjourney/Nano Banana для создания двух кадров: "X-ray/Exploded view" и "Assembled view".
3.  **Кодинг**: 
    - Используйте React + Three.js или GSAP для анимации скролла.
    - Реализуйте "ScrollStop" механику: видео или последовательность кадров проигрывается в зависимости от позиции скролла.
    - Обеспечьте Glassmorphism эффект для текстовых блоков поверх 3D.
4.  **Адаптивность**: Обязательно создавайте мобильную версию, где 3D модель корректно масштабируется.

## Пример Промпта
> "Используй скилл 3d_web_builder. Создай лендинг для логистической компании. Главный элемент — 3D фургон, который разбирается на части при скролле вниз. Стиль: Apple, минимализм."

- GSAP / Framer Motion
