# Запуск Claude Connector

Эта команда поднимет локальный сервер для работы с веб-версией Claude через AntiGravity.
Окно терминала нужно оставить открытым в фоновом режиме.

// turbo
```powershell
C:\ANTIGRAVITY\1\.venv\Scripts\python.exe C:\ANTIGRAVITY\1\scripts\claude_connector\claude_connector.py
```
