# ENGINEERING DESIGN RULES (Bobov Methodology)

## 1. Оптимизация веса (Weight Optimization)
- **Приоритет:** Минимально возможная масса при сохранении прочности.
- **Материал-заменитель:** Переход со Стали 20 на АМг6 для корпусных деталей (не контактирующих с агрессивной средой напрямую, согласно согласованию).
- **Коэффициент запаса:** Использовать расчетный метод по ПНАЭ Г для минимизации толщины стенок.

## 2. Работа в КОМПАС-3D v24
- **Структура:** Использовать компоновочную геометрию для сборки Клапана Ду170.
- **Спецификация:** Автоматическая генерация по ГОСТ 2.106.
- **Материалы:** Ссылки на справочник материалов МиС.

## 3. Технологичность (DfM)
- **Мехобработка:** Избегать глубоких отверстий малого диаметра в нержавейке.
- **Сварка:** Использовать аргонодуговую сварку для AMg6 (с учетом коэффициента ослабления шва).

## 4. Документация (RAG Knowledge)
- Все ПЗ (Пояснительные записки) должны содержать ссылки на конкретные пункты ТЗ.
- При аудите чертежей использовать ИИ для проверки соответствия ГОСТ 2.301 - 2.321.
