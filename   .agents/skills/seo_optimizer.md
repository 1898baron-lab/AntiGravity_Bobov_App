# Роль: MASTADONT SEO Analyst

## Описание
Этот скилл предназначен для превращения "просто кода" в "продукт, который находят в поиске". Основан на Видео #13 транскрипта.

## Инструкции для Агента
1.  **Аудит**: Проверьте текущий `index.html` или React-компоненты на:
    - Наличие тегов `<title>` и `<meta description>`.
    - Иерархию заголовков (один `<h1>`).
    - Атрибуты `alt` у изображений.
    - Канонические URL и OpenGraph теги.
2.  **Оптимизация**:
    - Генерируйте уникальные мета-теги на основе контента страницы.
    - Добавляйте Twitter Cards и JSON-LD разметку для Rich Snippets.
    - Настраивайте `sitemap.xml` и `robots.txt`.
3.  **Отчет**: После внесения правок выдайте таблицу "До/После" с оценкой по 100-балльной шкале (аналог Lighthouse).

## Пример Промпта
> "Используй скилл seo_optimizer для моего текущего проекта. Проведи полный аудит и исправь все ошибки в мета-тегах. Добавь OpenGraph для Telegram."

## Метрики успеха
- Score > 90 в Lighthouse.
- Наличие всех социальных превью.
- Семантическая корректность HTML5.
