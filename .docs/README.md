# External Documentation

Здесь хранится справочная документация, инструкции по библиотекам и любые материалы,
которые не должны индексироваться как база знаний проекта.

Примеры:
- README сторонних библиотек
- документация SDK
- вспомогательные статьи по инструментам

**Важно:** папка `.docs/` не входит в базу знаний локального RAG-агента.
