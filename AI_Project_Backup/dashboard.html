🛠️ Решение проблем
!

Ошибка: ModuleNotFoundError: No module named 'pywintypes'

Уже пофикшено — патч в mcp\os\win32\utilities.py. При обновлении пакета: uv tool install notebooklm-mcp-server --force и повтори патч.

!

Ошибка: The system cannot find the file specified

Значит .exe не установлен. Запусти: uv tool uninstall notebooklm-mcp-server && uv tool install notebooklm-mcp-server

!

Сессия истекла / NOT LOGGED IN

Запусти: C:\Users\Sasha Baron\.local\bin\notebooklm-mcp-auth.exe и войди в Google-аккаунт в открывшемся Chrome.

3

Установи зависимости

pip install aiogram
4

Запусти бота

python pdf_funnel_output/telegram_bot.py
