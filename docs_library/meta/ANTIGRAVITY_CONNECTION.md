## Что уже настроено
- В репозитории есть `obsidian_brain/_AI_EXCHANGE/TO_CHATGPT.md` и `FROM_CHATGPT.md` для обмена задачами и ответами.
- Есть `scripts/ai_watcher.py`, который может автоматически брать запросы из `TO_CHATGPT.md`, искать контекст в `obsidian_brain/Engineering` и отправлять их в Claude Connector.
- В архиве хранится `archive/mcp_config_backup.json` с примером конфигурации NotebookLM MCP.

## Как нам сконнектиться
1. Держи в синхронизации `obsidian_brain/_AI_EXCHANGE/TO_CHATGPT.md` для новых задач.
2. Отвечай через `obsidian_brain/_AI_EXCHANGE/FROM_CHATGPT.md`.
3. Для автоматизации запуска используй `scripts/ai_watcher.py` и локальный Claude Connector на `http://localhost:8765`.
4. Если есть MCP-сервер NotebookLM, проверь конфиг в `archive/mcp_config_backup.json` и настрой `notebooklm-mcp`.
