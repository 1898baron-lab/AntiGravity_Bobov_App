# Основы ЕСКД (Конструкторская документация)
*Подготовлено для проекта "Зевс" на основе требований Афанасьева А.И.*

## 1. Виды документов (ГОСТ 2.102)
В твоих наработках (Клапан Ду170) ты встретишь:
- **Чертеж детали (индекс отсутствует):** Изображение детали и данные для её изготовления. (Например: `Ствол.m3d`)
- **Сборочный чертеж (СБ):** Документ, содержащий изображение сборочной единицы. (`Клапан_сборка.a3d`)
- **Спецификация (СП):** Текстовый документ, определяющий состав сборочной единицы. **Это самый важный документ для инженера!**

## 2. Форматы (ГОСТ 2.301)
Основные размеры листов:
- **А4:** 210 x 297 мм
- **А3:** 297 x 420 мм
- **А2:** 420 x 594 мм

## 3. Основные надписи (ГОСТ 2.104)
То, что в народе называют "Штамп".
- **Графа 1:** Наименование изделия (например, "Клапан запорный").
- **Графа 2:** Обозначение документа (твой дешифратор/индекс).
- **Графа 3:** Материал (для деталей, например, "АМг6 ГОСТ 4784-97").

## 4. Особенности (Без электрики)
Фокусируемся на:
- Разрезах и сечениях (ГОСТ 2.305).
- Нанесении размеров и предельных отклонений (ГОСТ 2.307).
- Обозначении шероховатости поверхностей (ГОСТ 2.309).
