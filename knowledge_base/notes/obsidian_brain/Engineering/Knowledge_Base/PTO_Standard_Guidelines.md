# 📑 Регламент исполнительной документации (Стандарт MASTODONT)

На основе анализа руководства ПТО, для проекта "Зевс" и судебных экспертиз обязательны следующие документы:

## 1. Журналы учета
- **Общий журнал работ**: Хронология всех манипуляций на объекте.
- **Кабельный журнал**: Точный реестр соединений и длин (критично для ВВ-блоков Зевса).
- **Журнал сварочных работ**: Для проверки целостности рамных конструкций.

## 2. Акты проверок (Executive Acts)
- **АСР (Акт скрытых работ)**: Должен быть оформлен на заземление и внутренние кабельные трассы.
- **АООК**: Акт освидетельствования ответственных конструкций (высоковольтные узлы).

## 3. Технические требования
- Обязательное наличие **ППР (Проекта производства работ)** при любых перемещениях тяжелого оборудования.
- При отсутствии ППР любые работы (в т.ч. вывоз оборудования УК "Новоуральская") считаются технически незаконными.
