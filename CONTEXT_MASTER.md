## 3. ⚖️ ЮРИДИЧЕСКОЕ ДЕЛО БОБОВА (Статус: АКТИВНАЯ АТАКА)

### Цель
Переквалификация **ст. 330 УК РФ (Самоуправство)** → **ст. 158 ч. 4 (Кража в особо крупном размере)**

### Ключевые параметры
| Параметр | Значение |
|----------|----------|
| **Сумма иска** | 12 401 207 руб. |
| **Период волокиты** | 13 лет |
| **Тактика** | Жалобы по ст. 125 УПК РФ |
| **Доп. требование** | Компенсация по ФЗ № 68 (разумные сроки) |
| **Альт. статья** | ст. 293 УК РФ (Халатность должностных лиц) |

### Ключевые аргументы
- **Запрет:** Активы НИКОГДА не «мусор». Только **«Промышленные активы»** или **«Движимое имущество»**.
- Профессиональная оценка доказывает сумму > 1 млн руб. → особо крупный размер.
- 13 лет без решения = прямое нарушение ФЗ-68.
- Есть умысел на постоянное лишение → это Кража, а не Самоуправство.

### Доказательная база
- `legal\evidence\` — 23 фото и скана.
- `legal\` — жалобы, заявления, хроника.
- NotebookLM блокнот «AntiGravity_Bobov_Master» — Legal Corpus.

## 4. 🔧 ИНЖЕНЕРНЫЙ ПРОЕКТ «ЗЕВС» (Статус: В РАБОТЕ)

### Проект: Клапан DN170 (Росатом / ЦПТИ)

| Параметр | Значение |
|----------|----------|
| **Текущий вес** | ~65 кг (сталь) |
| **Целевой вес по ТЗ** | 25 кг |
| **Целевой вес уточнённый** | 22.15 кг |
| **Материал (новый)** | АМг6 (плотность ~2.65 г/см³) |
| **Стандарты** | NP-016-05, NP-089-15, ПНАЭ Г |
| **Инструмент** | КОМПАС-3D v24 |
| **Наставник** | Афанасьев Андрей Игоревич (Красноярск) |

### Методология ЕСКД
- Строгое соответствие ГОСТ (ЕСКД серии 2.3хх).
- Аргонодуговая сварка для АМг6 (учёт коэффициента ослабления шва).
- Расчётный метод ПНАЭ Г для минимизации толщины стенок.
- Автогенерация спецификации по ГОСТ 2.106.

### Проект: Сборочно-сварочный стапель "ЗАКРЕП v1.0" (ЦПТИ / Росатом)
- **Цель стажировки:** 01.04.2026 – 25.12.2026 в качестве Инженера-конструктора II категории по оснастке (ССО).
- **Задача:** Разработка сквозного пакета КД/ТД на стапель для рамы (1200x800 мм) из профильной трубы 40х40х3 (S235/Ст3).
- **Методология сварки:** ГОСТ Р ИСО 2553 (обозначения швов), ГОСТ Р ИСО 15609-1 (WPS/pWPS), качество ISO 5817, допуски ISO 2768-1 (m).
